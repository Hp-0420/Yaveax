How do I install UPS systems?

UPS installation is straightforward:

  1. Location: Place in well-ventilated, dry area
  2. Battery: Connect battery terminals (red to positive, black to negative)
  3. Input: Connect main power supply to input socket
  4. Output: Connect devices to UPS output sockets
  5. Testing: Switch on and test battery backup

Professional installation service available for complex setups.

What if my LED bulb is flickering?

LED flickering can be resolved by:

  • Check voltage: Ensure stable 220-240V supply
  • Switch compatibility: Use LED-compatible switches
  • Loose connections: Tighten all electrical connections
  • Dimmer issues: Use dimmable LEDs with compatible dimmers

If problem persists, contact our technical support for assistance.
